# ⚖️ Landing Page - Advocacia Especializada

Esta é uma landing page moderna e responsiva desenvolvida para o setor jurídico. O projeto foi criado com o objetivo de demonstrar habilidades em desenvolvimento web, design focado em conversão (UX/UI) e organização de código.

---

## 🚀 Propósito do Projeto
> **Nota:** Este projeto foi desenvolvido exclusivamente para fins de **portfólio pessoal**. 
> As informações, nomes e contatos contidos aqui são fictícios, servindo apenas para ilustrar a estrutura de um site profissional para advogados.

---

## 🛠️ Tecnologias Utilizadas
Para a construção deste projeto, utilizei as seguintes tecnologias:

* **HTML5** - Estrutura semântica.
* **CSS3** - Estilização moderna e layout responsivo.
* **JavaScript** - Interatividade e animações (se houver).
* **Git & GitHub** - Controle de versão e hospedagem do código.

---

## 📱 Funcionalidades de Destaque
* **Design Mobile-First:** Totalmente otimizado para celulares e tablets.
* **Seções Estratégicas:** Áreas de atuação, sobre o profissional, depoimentos e FAQ.
* **Chamadas para Ação (CTA):** Botões estratégicos para contato via WhatsApp ou formulário.
* **Performance:** Carregamento rápido e código limpo.
